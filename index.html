Propuesta de solución para la prueba técnica

En el proceso de selección se pide realizar una prueba técnica usando como herramienta la plataforma RunMyProcess, para ello se necesita crear una cuenta de prueba en donde es necesario esperar a que el soporte de RunMyProcess te contacte via email.
Al día de hoy no se ha realizado ningún contacto y no he podido acceder a la plataforma.
Es por eso que haciendo un llamado a mis ganas de avanzar, decido crear este sitio usando HTML, CSS y Javascript como herramientas para mostrar la solución que propongo al problema planteado.
Yo supongo que es la herramienta que usará la persona seleccionada para este cargo y por lo que he revisado es muy intuitiva y con una buena documentación para que aprender a usarla sea muy fácil y rápido.
